📝 Methodology

Data Source: NBA API (PlayByPlayV3) — real play-by-play data from multiple NBA seasons.
Timeout Definition: Non-official (team-called) timeouts identified from play-by-play descriptions.
Features Captured: For each timeout: opponent scoring run, own scoring run, score differential, quarter, clock time, FG%, turnovers.
"Beneficial" Label: A timeout is labeled beneficial if (a) the team's score differential improved in the next 15 plays, OR (b) a 7+ point opponent run was stopped to ≤2 points after.
Statistical Tests: Mann-Whitney U tests for group comparisons. Pearson correlation for continuous relationships. p < 0.05 considered significant.
Simulator: Uses kernel density matching against historical timeouts in similar game states to estimate success probability.
